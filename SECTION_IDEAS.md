# 📋 Section Ideas - Adaptasi dari AFU Nagara ke Wedding Invitation

## Mapping Section dari AFU ke Wedding

| AFU Section | Wedding Equivalent | Description |
|-------------|-------------------|-------------|
| Opening | Opening | Hero section dengan nama pengantin |
| Salam | Salam | Greeting & profil pengantin |
| Event | Event | Detail waktu & tempat acara |
| Maps | Maps | Lokasi venue dengan Google Maps |
| ToR | Gallery/Love Story | Gallery foto atau timeline cinta |
| Rangkaian 10 Kota | - | (Tidak relevan untuk wedding) |
| Narasumber | Gift/Hadiah | Info rekening/hadiah |
| Pengisi Acara | Wedding Party | Keluarga & wedding party |
| Tamu Undangan | Guest Book | Ucapan & doa dari tamu |
| Ketentuan | Protokol Kesehatan | Himbauan untuk tamu |
| RSVP | RSVP | Form konfirmasi kehadiran |
| Thanks | Thanks | Terima kasih & penutup |

## Section yang Akan Ditambahkan

### 1. Gallery Section
**Posisi:** Setelah Salam, sebelum Event
**Konten:**
- Carousel/Grid foto pre-wedding
- Video pre-wedding (optional)
- Background music player

### 2. Love Story Timeline
**Posisi:** Setelah Gallery
**Konten:**
- Timeline perjalanan cinta
- Dari bertemu sampai tunangan
- Milestone penting

### 3. Gift/Hadiah Section
**Posisi:** Setelah Maps
**Konten:**
- Nomor rekening untuk hadiah
- QR Code payment
- Alamat pengiriman hadiah fisik

### 4. Wedding Party Section
**Posisi:** Setelah Gift
**Konten:**
- Orang tua pengantin
- Saksi nikah
- Best man & bridesmaids
- Wedding organizer

### 5. Guest Book Section
**Posisi:** Setelah RSVP
**Konten:**
- List ucapan & doa dari tamu
- Fitur like/love
- Real-time updates

### 6. Protokol Kesehatan
**Posisi:** Sebelum RSVP
**Konten:**
- Himbauan protokol kesehatan
- Dress code
- Larangan membawa anak kecil (jika ada)

### 7. Live Streaming Info
**Posisi:** Di Event section
**Konten:**
- Link YouTube/Instagram Live
- Jadwal live streaming
- Cara menonton
